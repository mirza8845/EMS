# Employee Management System (EMS)

A modern and scalable Employee Management System (EMS) designed to streamline employee administration, attendance, payroll, department management, and organizational workflows. The application provides an intuitive interface for administrators and employees while ensuring secure authentication, efficient data management, and a responsive user experience.

---

## 📌 Overview

Employee Management System (EMS) is a comprehensive solution for managing an organization's workforce. It centralizes employee information, improves operational efficiency, and reduces manual administrative work through automation.

The system enables administrators to manage employees, departments, attendance, payroll, and employee records while providing employees with access to their own information.

# Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/EMS.git
```

```bash
cd EMS
```

---

## Install Dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

---

## Environment Variables

Create a `.env` file inside the server directory.

Example:

```env
PORT=5000

MONGODB_URI=your_database_url

JWT_SECRET=your_secret_key
```

---

## Run Project

### Backend

```bash
npm run dev
```

### Frontend

```bash
npm start
```
